# An autonomous self-driving laboratory for accelerated electrolyte discovery
In this work, we developed an autonomous self-driving laboratory (SDL) and used the SDL for the Optimization of Electrolytes for low-temperature zinc-ion batteries.

This repository contains supplementary materials for the paper [An autonomous self-driving laboratory for accelerated electrolyte discovery]().



**`code`**: Code that was used in the paper.



**`source data`**: source data of the paper.



**`3D Printing Files`** *STL* file for the electrode holders.
